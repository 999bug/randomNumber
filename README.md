# 🎲 随机数生成器

用于教学场景的随机数生成工具，按设定的出错率自动注入错误数据。

## 功能

- 三种生成模式：**整数** / **小数** / **比值**（1:N）
- 可设定范围、数量、小数位数、出错率
- **错误数互不相邻**：任意两个错误之间至少间隔 3 个正确数
- **错误数标识**：页面显示红色文字 + 黑色下划线 + 浅黄底
- **一键复制**：富文本复制，粘贴到 Word/WPS 中错误数自带下划线
- 响应式布局，手机/电脑自适应

## 使用方法

1. 浏览器打开 `随机数V4.0.html`
2. 填写参数：
   - **出错率**：错误数据占比（0-100%）
   - **范围**：随机数取值范围
   - **随机数数量**：生成个数（最多 500）
   - **小数位数**：0 为整数，1-7 为指定位小数
3. 点击生成按钮：输出整数 / 输出小数 / 输出比值
4. 点击 **📋 一键复制** 粘贴到文档

## 技术信息

- 纯 HTML + CSS + JavaScript
- 零外部依赖，浏览器直接打开即用
