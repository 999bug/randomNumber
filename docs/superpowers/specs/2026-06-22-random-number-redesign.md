# 随机数生成器重设计

## 目标

基于现有 `随机数3.7.html` 的功能，创建全新的 `随机数4.0.html`，改进 UI 和交互体验。

## 功能需求

### 保留现有功能
1. 输入参数：出错率(%)、范围(小/大)、随机数数量、小数位数
2. 三种生成模式：输出整数、输出小数、输出比值
3. 错误注入：按出错率在随机位置注入错误值（整数模式下±3偏差，小数模式下超出范围±0.2）
4. 统计显示：出错数量、合格数量、合格率
5. 重置 / 重置全部

### 新增功能
1. **错误不挨着**：任意两个错误位置之间至少间隔 3 个正确数。错误率过高无法满足时自动降低并提示
2. **下划线标识**：错误数使用红色下划线标识
3. **一键复制**：复制所有数值（纯数字逗号分隔），点击后短暂显示"✓ 已复制"

### 技术改进
1. 动态生成输出 DOM，不再硬编码 104 个 span
2. 错误分布算法：先确定有效位置集合，从中选取而非随机碰撞
3. 不使用任何外部 CSS/JS 框架，单文件自包含

## 布局

左右双栏布局：
- 左栏（~35%）：控制面板，包含输入框和操作按钮
- 右栏（~65%）：输出区，数字网格 + 统计信息 + 复制按钮

## 视觉设计

- 白色背景 + 圆角卡片 + 微妙阴影
- 蓝色主色调按钮（`#4A90D9`）
- 数字使用等宽字体（`ui-monospace`）
- 正确数：深色文字，浅灰底卡片
- 错误数：红色文字 + 黑色下划线，浅红底卡片
- 响应式：宽度＜768px 时切换为上下布局

## 实现范围

- 单个文件 `随机数4.0.html`
- 不改动 `随机数3.7.html`
- 不添加外部依赖
